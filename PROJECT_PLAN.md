# 프로젝트: 우리 동네 생활 정보 웹사이트

## 목표
공공데이터포털(data.go.kr)에서 지역 행사/축제/지원금 정보를 자동 수집하고,
AI가 매일 블로그 글을 자동 작성하며,
Google AdSense + 쿠팡 파트너스로 수익화하는 웹사이트

## 기술 스택
- **Next.js (App Router)**: 웹사이트를 만드는 최신 도구 (TypeScript 포함)
- **Tailwind CSS**: 디자인을 쉽고 예쁘게 꾸며주는 도구
- **Gemini API**: 구글의 인공지능으로 블로그 글을 자동으로 써주는 역할
- **공공데이터포털 API**: 정부나 지자체에서 제공하는 실시간 정보를 가져오는 통로
- **GitHub Actions**: 매일 정해진 시간에 컴퓨터가 알아서 일을 하도록 만드는 자동 스케줄러
- **Cloudflare Pages**: 만든 웹사이트를 인터넷에 무료로 올려주는 서비스

## 페이지 구성
1. **메인 페이지**: 이번 달 행사/축제 + 지원금/혜택 정보를 카드 형태로 한눈에 보기
2. **상세 페이지**: 특정 행사나 혜택을 눌렀을 때 나오는 자세한 설명
3. **블로그 목록**: AI가 자동으로 작성한 유용한 정보 글 리스트
4. **블로그 상세**: 개별 블로그 글 본문

## 수익화 (돈 버는 방법)
- **Google AdSense**: 웹사이트 곳곳에 구글 광고를 배치하여 클릭 수익 발생
- **쿠팡 파트너스**: AI가 쓴 글 아래에 관련 상품 링크나 배너를 달아 구매 수수료 발생

## 자동화 (GitHub Actions)
매일 아침 7시(한국시간)에 자동으로 다음 단계가 실행됩니다:
1. 공공데이터 API에서 새로운 소식을 가져옵니다.
2. Gemini AI가 그 소식을 바탕으로 친절한 블로그 글을 씁니다.
3. 바뀐 내용을 저장소(GitHub)에 올립니다.
4. Cloudflare Pages가 자동으로 웹사이트를 최신 상태로 바꿉니다.

## 환경변수 (비밀번호 같은 중요 정보)
- `GEMINI_API_KEY`: 인공지능 사용권한 키
- `PUBLIC_DATA_API_KEY`: 공공데이터 사용권한 키
- `NEXT_PUBLIC_ADSENSE_ID`: 구글 애드센스 광고판 ID
- `NEXT_PUBLIC_GA_ID`: 구글 분석기(방문자 체크) ID
